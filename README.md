
# 🍲 RecipeBuzz — Authentic Pakistani Cuisine Finder

RecipeBuzz is a beautiful, fully responsive front-end web application designed to help users discover, filter, and save authentic Pakistani recipes. From Halwa Puri to Beef Nihari, every dish tells a story.

## ✨ Features

* **Dynamic Recipe Grid:** Displays high-quality images, preparation times, and categories for 10 authentic Pakistani dishes.
* **Smart Search & Filtering:** Instantly search for recipes by name or ingredient, or filter by category (Breakfast, Lunch, Dinner, Dessert, Snack).
* **Favorites System:** Save your favorite recipes to a dedicated "Favorites" section. Data is persisted using the browser's `localStorage`, so you won't lose your saved dishes when you refresh the page.
* **Detailed Recipe Modals:** Click on any recipe to open a sleek pop-up modal containing full ingredients, step-by-step instructions, and local customer reviews with star ratings.
* **Quick Instructions Toggle:** Expandable accordion right on the recipe card for a quick glance at the cooking steps without opening the full modal.
* **Smooth Animations:** Custom CSS keyframe animations for card hover effects, modal slide-ins, and a rotating logo.
* **Fully Responsive:** Beautifully optimized for both desktop and mobile screens.

## 🛠️ Technologies Used

* **HTML5:** Semantic markup for structure.
* **CSS3:** Custom CSS variables (variables), Flexbox, CSS Grid, media queries, and pure CSS animations (no external CSS libraries).
* **Vanilla JavaScript (ES6+):** DOM manipulation, array filtering, event listeners, and `localStorage` management without the need for heavy frameworks like React or jQuery.

## 🚀 Getting Started

Since this project uses pure HTML, CSS, and JavaScript, no build tools or package managers (like npm) are required.

### Prerequisites
A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation
1. Clone the repository to your local machine:
   ```bash
   git clone [https://github.com/yourusername/recipebuzz.git](https://github.com/yourusername/recipebuzz.git)

```

2. Navigate into the project directory:
```bash
cd recipebuzz

```


3. Open `index.html` in your browser. You can do this by double-clicking the file or using a local development server like VS Code's "Live Server" extension.
